## 🚀 Run Locally

### Smart Contracts

```bash
  cd smartcontract
  npx hardhat compile
  npm npx hardhat run scripts/deploy.js --network testnet
```

### Dashboard

Create an environment file and provide it with Pinata JWt

```
PINATA_JWT=<your_pinata_jwt>
```

```bash
  cd Dashboard
  npm i
  npm run dev
```

### Marketplace

```bash
  cd 3D-Marketplace
  npm i
  npm run start
```
